# m

#### 介绍
一个授权管理系统，此仓库为后端代码

#### 配套前端

暂无

#### 开发环境
- Windows10 家庭版 21H1
- Java JDK 11
- Maven 3.6.3
- IDEA 2021.1
- Redis 3.0.504
- MySQL 8.0.12


#### 安装教程

1.  克隆仓库到本地
2.  修改配置：src/main/resources/application-dev.yml
3.  启动Redis、MySQL
4.  运行MyAuthApplication.java启动类
5.  访问http://localhost:8081/myauth/web/connect查看是否正常
